## üìÖ 3-Week Prototype Development Plan

### Week 1: Chair Prototype Development

#### Day 1-2: Frame Fabrication
- **Cut side frame components** using existing 25√ó25mm tube + 30√ó6mm bar
- **Weld connection points** for M8 bolts and centering pins
- **Test fit** all components before final welding
- **Surface preparation** - degreasing and mechanical roughening

#### Day 3-4: Assembly System Integration  
- **Install welded nuts** at connection points
- **Drill centering pin holes** with positioning jigs
- **Create slot holes** for painting tolerance accommodation
- **Test assembly** with actual M8 hardware

#### Day 5-7: Finishing & Testing
- **Apply anti-corrosion primer** (critical step)
- **Apply 2K top coat** (2 coats minimum)
- **Assembly test** - time first assembly from scratch
- **Disassembly test** - verify no damage to connections
- **Stacking test** - verify 6-chair stack stability

#### Week 1 Deliverables
- ‚úÖ Complete chair prototype
- ‚úÖ Assembly time documentation
- ‚úÖ Stacking verification photos
- ‚úÖ Hardware compatibility confirmation

### Week 2: Transport & Packaging Validation

#### Day 8-9: Packaging Development
- **Design flat-pack layout** for 6-chair configuration
- **Source packaging materials** (foam, cardboard, stretch film)
- **Create packing procedure** with photo documentation
- **Test packaging** with actual prototype

#### Day 10-11: Transport Simulation
- **Drop test** from 1m height (package level)
- **Vibration test** - simulate truck transport
- **Stack compression test** - 6-chair stack weight
- **Edge impact test** - corner protection verification

#### Day 12-14: Cost Analysis
- **Hardware cost calculation** per unit

## üéØ Quality Control System

### Incoming Material Inspection
- **Steel tubing**: Dimension check (¬±0.5mm), surface quality
- **Flat bar**: Straightness, thickness verification
- **Hardware**: Thread quality, corrosion resistance
- **Wood components**: Moisture content, dimensional accuracy

- **Drilling**: Hole position accuracy (¬±0.2mm)
